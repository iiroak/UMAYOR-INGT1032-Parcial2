# UMAYOR-INGT1032-Parcial2

Repositorio de evaluación para **INGT1032** con trabajo en **R** sobre análisis exploratorio de datos, acompañado de archivos R Markdown, PDFs y una planilla de datos.

## Qué contiene

- **Prueba.R**: script con análisis exploratorio de variables como categoría, subcategoría y descuento.
- **Sample - Superstore.xls**: base de datos principal usada en el análisis.
- **Pregunta1.Rmd**, **Pregunta12.Rmd**, **Pregunta13.Rmd**: documentos de desarrollo por pregunta.
- **Pregunta1.pdf**, **Pregunta12.pdf**, **Pregunta13.pdf**: salidas exportadas.
- **Prueba.Rproj**: proyecto de RStudio.
- **22222222222222.pdf**: archivo PDF adicional presente en la raíz.

## Enfoque del repositorio

El script visible trabaja con:

- lectura de la hoja `Orders` desde Excel,
- selección de variables relevantes,
- resúmenes descriptivos,
- tablas de frecuencia,
- gráficos básicos con `ggplot2`.

## Cómo revisarlo

Abrir el repositorio en RStudio o ejecutar el script principal desde R:

```r
source("Prueba.R")
```

Si se quiere reconstruir parte del trabajo, revisar también los `.Rmd` y los `.pdf` de cada pregunta.

Este repositorio parece reunir materiales de una evaluación parcial, con evidencia tanto del código como de los documentos entregados.
